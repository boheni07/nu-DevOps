# 보안 체크리스트 (Security Checklist)

엔터프라이즈 레벨의 보안 신뢰성을 확보하기 위한 점검 항목입니다.

## 1. 인증 및 권한 (Identity & Auth)
- [ ] 패스워드는 반드시 안전한 해시 함수(Bcrypt 등)로 서버에서 암호화되어 저장되는가?
- [ ] API 접근 시 유효한 JWT 토큰을 매번 검증하는가?
- [ ] 권한이 없는 사용자가 관리자 페지에 직접 경로를 입력하여 접근할 수 없는가?

## 2. 데이터 보호 (Data Protection)
- [ ] API 통신 결과에 민감한 개인정보(이메일 외 패스워드 등)가 포함되어 있지는 않은가?
- [ ] 외부 라이브러리의 취약점을 주기적으로 점검(npm audit)하고 최신 버전으로 유지하는가?
- [ ] API Key 등 외부 인터페이스 키가 클라이언트 사이드 코드에 노출되지 않았는가?
- [ ] 보고 완료 및 승인된 데이터에 대해 서버/클라이언트 측에서 수정 시도를 차단하는가?
- [ ] 특정 사용자의 리포트가 권한이 없는 타인에게 노출되지 않도록 필터링되고 있는가?

## 3. 네트워크 및 서버 (Network & Server)
- [ ] 모든 트래픽은 HTTPS(TLS 1.2 이상)를 통해서만 전송되는가?
- [ ] 관리 주체가 불분명한 포트가 오픈되어 있지는 않은가?
