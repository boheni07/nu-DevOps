# 03. 사용자 가이드 (User Guide)

## 1. 대시보드 (Dashboard)
- 시스템에 접속하면 가장 먼저 표시되는 화면입니다.
- **프로젝트 현황**: 진행 중인 프로젝트의 상태, 일정, 진행률을 요약하여 보여줍니다.
- **나의 업무**: 로그인한 사용자에게 할당된 업무 목록을 확인할 수 있습니다.

## 2. WBS 관리 (Work Breakdown Structure)
- 프로젝트의 업무를 계층적으로 관리하는 화면입니다.
- **새 업무 등록**: '새 업무 등록' 버튼을 클릭하여 새로운 최상위 업무를 생성할 수 있습니다.
- **하위 업무 추가**: 각 업무 행의 우측 'Actions' 컬럼에서 `+` 아이콘을 클릭하여 하위 업무를 추가합니다.
- **순서 변경 및 계층 이동**: 업무 행의 좌측 이동 핸들(아이콘)을 드래그하여 순서를 변경하거나, 다른 업무 하위로 이동시킬 수 있습니다.
- **상태 자동 계산**: 하위 업무의 일정이나 진척률이 변경되면, 상위 업무에 자동으로 반영됩니다.

## 3. 간트 차트 (Gantt Chart)
- 프로젝트 일정을 시각적으로 확인하는 화면입니다.
- **주간/일간 보기**: 우측 상단의 버튼을 통해 보기 모드를 전환할 수 있습니다. (기본값: 주간)
    - **주간(Week)**: 전체적인 일정 흐름을 파악하기 좋습니다.
    - **일간(Day)**: 상세한 일정을 확인할 때 유용합니다.
- **자동 맞춤**: 브라우저 창 크기에 맞춰 차트 너비가 자동으로 조절되므로, 가로 스크롤 없이 전체 일정을 한눈에 볼 수 있습니다.
- **범례**:
    - **회색 막대**: 계획된 일정 (Planned)
    - **유색 막대**: 실제 진행 일정 (Actual). 상태(진행 중, 완료, 지연 등)에 따라 색상이 다릅니다.

## 4. 내 업무 보드 (My Tasks)
- 칸반 보드(Kanban Board) 형태의 업무 관리 도구입니다.
- **To Do / In Progress / Done**: 업무 카드를 드래그하여 상태를 변경할 수 있습니다.

## 5. 성과 리포트 및 업무보고 (Progress Reporting)
- 실적 집계 및 공식 업무 보고를 수행하는 화면입니다.
- **보고서 생성**: '일간/주간/월간' 중 리포트 타입을 선택하고 프로젝트/날짜를 지정하여 생성 버튼을 클릭합니다.
- **업무 내역 확인**: 선택된 기간의 업무 로그가 실적 테이블에 자동으로 표시됩니다.
- **보고 및 제출**: 종합 의견을 작성한 후 '보고하기(Submit)' 버튼을 클릭하여 PM에게 제출합니다. 제출된 리포트는 수정이 불가능합니다.
- **이력 관리**: 화면 하단의 이력 섹션을 통해 과거 보고서를 즉시 조회할 수 있습니다.
- **보고서 취합(PM 전용)**: 팀원들이 제출한 보고서를 모아보고 '최종 승인' 처리를 수행합니다.

## 6. 관리 기능 (Management)
- **Project Manager**: 프로젝트를 생성하거나 수정합니다.
- **Resource Optimizer**: 팀원별 업무 배분 현황을 확인하고 과부하를 방지합니다.
- **Member Manager**: 팀원을 초대하거나 정보를 수정합니다.
