# 05. 변경 이력 (Changelog)

## [Unreleased]

### Added
- **문서화**: `/docs` 폴더 내 모든 문서 최신화 (아키텍처, 데이터 모델, 변경 이력 등).
- **데이터 관리**: `DataManagement` 컴포넌트를 통한 JSON 백업/복원 및 샘플 데이터 로딩 기능 강화.

### Changed
- **프로젝트 관리 UI**:
    - 프로젝트 카드에서 'Workstation Switch' 기능을 제거하고 'Edit', 'Delete' 버튼을 강조하여 UI 직관성 개선.
    - 프로젝트 생성/수정 모달의 레이아웃을 컴팩트하게 조정하고 멤버 선택 로직(Client, PM, PL) 강화.
- **성과 리포트 및 업무보고**:
    - 리포트 상세 뷰 컴포넌트를 더욱 컴팩트하게 조정하여 정보 밀도 향상.
- **전역 레이아웃**:
    - 사이드바 및 헤더 영역의 디자인 고도화 (그라데이션, 애니메이션 효과 추가).

### Fixed
- **안정성**: Gemini API 키 누락 시 앱이 크래시되는 현상을 방지하고 폴백 메시지를 출력하도록 수정.

## [1.0.0] - 2025-02-12

### Added
- **간트 차트**: 주간 보기 모드 및 화면 자동 최적화 기능.
- **Gemini AI 연동**: `gemini-2.0-flash` 모델 기반 프로젝트 브리핑 및 리소스 최적화 제안.
- **WBS**: 드래그 앤 드롭 업무 관리 및 상위 업무 상태 자동 계산.
- **성과 리포트**: 일간/주간/월간 보고서 및 승인 프로세스.
