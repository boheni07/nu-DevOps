# nu-DevOps
### Next-Gen SW Collaboration Platform

**nu-DevOps**는 소프트웨어 개발 프로젝트의 전 과정을 효율적으로 관리하고 Gemini AI를 통해 협업을 극대화하는 차세대 DevOps 플랫폼입니다.

---

## 🚀 주요 기능

-   **AI-Powered Insights**: Gemini AI를 활용한 프로젝트 진행 보고서 생성 및 리소스 최적화 제안.
-   **Dynamic Project Hub**: WBS 관리 및 실시간 일정 시각화(Gantt Chart).
-   **Integrated Workspace**: 개인화된 칸반 보드 및 성과 분석 리포트.
-   **Advanced Management**: 이용 기관, 회원 및 프로젝트의 통합 관리 시스템.

## 🛠️ 시작하기 (Quick Start)

**사전 요구사항:** [Node.js](https://nodejs.org/) (v18 이상)

1. **의존성 설치:**
   ```bash
   npm install
   ```
2. **환경 변수 설정:**
   `.env.example` 파일을 복사하여 `.env` 파일을 만들고 `GEMINI_API_KEY`를 설정하세요.
   ```bash
   GEMINI_API_KEY=your_api_key_here
   ```
3. **개발 서버 실행:**
   ```bash
   npm run dev
   ```
